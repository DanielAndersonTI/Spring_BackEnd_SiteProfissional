# Spring Back-End - Site Profissional

Este é o repositório do back-end do site profissional desenvolvido com Spring Boot. Ele serve como a API para a parte frontal do site, fornecendo funcionalidades como o gerenciamento de dados e integração com o banco de dados.

## Tecnologias Utilizadas

- **Spring Boot**: Framework para desenvolvimento do back-end.
- **Java 11+**: Linguagem utilizada para o desenvolvimento da aplicação.
- **Maven**: Ferramenta para gerenciamento de dependências e execução de builds.
- **MySQL**: Banco de dados utilizado para persistência de dados.

## Funcionalidades

- **Cadastro de Usuários**: Permite o cadastro de usuários para acessar o site.
- **Envio de Contato**: Formulário de contato onde os dados são recebidos e processados.
- **Integração com Front-End**: API RESTful para comunicação com o front-end em React.

## Como Rodar o Projeto

### Pré-requisitos

- **Java 11 ou superior**: Baixe e instale o JDK em [https://adoptopenjdk.net/](https://adoptopenjdk.net/)
- **Maven**: Certifique-se de ter o Maven instalado. Caso contrário, baixe e instale em [https://maven.apache.org/install.html](https://maven.apache.org/install.html)
- **MySQL**: Configure o MySQL no seu computador ou use um serviço de banco de dados.
